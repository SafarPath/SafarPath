## 🔧 For Developers

### Local Development
If you want to run the website locally:

```bash
# Clone the repository
git clone https://github.com/SafarPath/SafarPath.git

# Navigate to the project
cd SafarPath

# Open in your preferred code editor
code .
```

### Project Structure
